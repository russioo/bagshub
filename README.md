# BagsHub

Token tracker for Bags on Solana.

## Features

- Token list with price, volume, market cap
- Token detail pages
- Token launcher (via Bags API)
- Search

## Setup

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Bags API

Get your API key from [dev.bags.fm](https://dev.bags.fm)

```env
BAGS_API_KEY=your_key
```
